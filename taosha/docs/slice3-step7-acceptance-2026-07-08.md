### 10.1 口径定义(pap cost 冻结;骗不了人的算术读法)
- **进场** = τ=0(=T+1,含一字板顺延/放弃 by cleaning `MAX_POSTPONE=5`)当日**后复权 open**;
- **出场** = 检验窗尾(主窗 τ=main_len−1=+19 / 稳健窗 τ=robust_len−1=+59)当日**后复权 close**;
- **一字涨停不可成交**(`limit_up_board_untradeable`)= 进场侧,已由 cleaning 顺延/放弃处置;
- **成本乘式净额**(cost 冻结率的算术读法,⚠ 请人于开卡确认此读法即封存所对口径):
  - 买费 = 佣金 + 滑点 = 0.00025 + 0.001 = **0.00125**
  - 卖费 = 佣金 + 印花(卖) + 滑点 = 0.00025 + 0.001 + 0.001 = **0.00225**
  - `净额 = close_exit·(1 − 卖费) / (open_entry·(1 + 买费)) − 1`;毛额 = `close_exit/open_entry − 1`。
- **⚠ 出场侧不可成交**(窗尾一字跌停/停牌)pap **未定**顺延规则 → **不发明**:窗尾 close 存在即按字面
  "窗尾收盘出"计、单列 exit 诊断计数(一字/涨跌停);窗尾停牌**缺 close** → 该事件该窗**排除**、计
  `excluded_no_close`(保守,不以不可成交价充数)。**可交易口径为报告与锚对照用途,不改统计判决。**
- **⚠ 口径性质**:净额=**原始持有净收益**(含市场 β 漂移),**非**异常收益;与 §9 统计口径 CAAR
