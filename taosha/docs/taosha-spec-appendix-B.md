# 淘沙施工 spec v0.2 · 附录B(append,2026-07-05 人批)

**B1 · 假设 #2"策略版"的操作化定义(消除 spec §1"不做策略级回测"与 §6"策略版"的表述冲突):**

#2 策略版 = **单事件持有路径模拟**:每个进场事件独立模拟一条持仓路径——按冻结离场规则(成本价−20% 强平 或 收盘破20日线,先到先出)逐日推进至离场,记录该路径的净收益(成本全额扣减)。各事件路径相互独立:**无组合构建、无资金约束、无仓位重叠处理、无再平衡**——输出为"事件级净收益分布"及其检验,不是账户级净值曲线。此定义仍在事件研究范式内,不触发"策略级组合回测"禁区(该禁区指含资金/仓位/组合逻辑的账户级回测,触发条件不变)。

**B2 · 检验口径**:策略版收益同样过 ADJ-BMP(以路径净收益替代固定窗 CAR)、样本量闸与门槛校正;同一事件同时输出事件版(固定窗)与策略版两份结果,分别入 result_json,互为体检。
