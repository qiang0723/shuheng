## F1 · 逐条冻结(九点)

1. **H60 与回撤触发**:H60 = 以评估日为末端的 **trailing-60 交易日后复权收盘最高价**(逐日滚动);
   回撤触发 ⇔ `close ≤ 0.9×H60`。

2. **均线**:ma10 / ma20 = **trailing-10 / trailing-20 交易日后复权收盘 SMA**(非 EMA)。

3. **站上(水平判据)**:`close > ma10`;armed 后首个满足(`close > ma10`)之日起算连续计数。

4. **连续 3 日**:连续 3 个交易日 `close > ma10`,**第 3 日 = 进场日 T**;期间任一日 `close ≤ ma10`
   计数归零(armed 内重计)。**等号归"破"**(`close = ma10` 视为不满足 `close > ma10`,即破)。

5. **破 20 日线失效**:armed 期内任一日 `close < ma20` → **失效** = 解除 armed + 计数清零,
   须新一轮回撤方可重新 armed。**效力域限进场前**;进场后的"破 20 日线"属**策略版出场规则
   (附录 B 域)**,状态机分开;**事件版检验窗按 pap 20/60 固定窗,不受出场影响**。

6. **armed 无超时**:终止仅由"进场"或"失效"二者先到(**不引入自由参数 / 时窗**)。

7. **同票多次进场(逐次独立事件)+ #7 补丁**:允许同票多次进场。
   **"新一轮"操作化(补丁)**:进场或失效之后,须**至少存在一日 `close > 0.9×H60`(脱离回撤区)**,
   此后再度 `close ≤ 0.9×H60` 方构成新一轮 armed。理由 = 防"进场点仍在回撤区内导致 armed 立即
   重挂"的链式病灶;H60 滚动使"脱离"自然可达,**无新参数**。

8. **事件锚**:T = 进场日(收盘确认);**τ=0 := T+1**(S2-DEC3 τ 轴,盘后信号前视规避)。

9. **PIT 强制**:全部指标(H60 / ma10 / ma20)**仅用评估日及之前数据**,无前视。
