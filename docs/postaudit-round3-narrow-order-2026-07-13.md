## #1-b 代理规则拟冻结文本(能力边界声明,随验收档交人批)

**next_open 离场成交之"开盘时点可成交"代理规则(日线代理,非真实委托成交验证):**

判定仅消费开盘时点(9:25 集合竞价撮合后)已可得的信息:

- **R-open-1(opening print 存在)**:该日存在 present bar 且开盘价有值(>0)
  = 当日市场发生过开盘竞价成交。**已知局限:市场存在成交 ≠ 我方委托单能排上队成交,
  本条不证明排队优先级。**
- **R-open-2(开盘价不在跌停价位)**:原始开盘价 ≠ round(原始前收 × (1 − limit_pct), 2)
  (limit_pct 复用 qbase 视图既有口径:主板 10% / ST 5% / 创业板 2020-08-24 后 20% /
  科创板 20%,分位取整)。开盘价恰在跌停价位 → 视为卖单堆积、按不可成交处理(保守);
  开盘价在跌停价位之上 → 视为可成交。**已知局限:开盘价高于跌停位不保证我方卖单在
  开盘竞价中成交;本条是价格位置代理,不是成交回报验证。**

"已验证真实可成交"。** 日终字段(收盘 limit_status / one_word / close)一律不得进入
