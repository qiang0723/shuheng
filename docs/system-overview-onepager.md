## 你在盖什么

一套 1–3 人的人机结合投研体系,两年验证"人的认知判断有没有真 alpha"。全部纪律不保证有 edge,只保证诚实地知道有没有。**红线用结构焊死(触发器/只读账号/冻结配置),不靠自觉**——这是你读到的一切"怪规矩"的共同理由。

## 四层架子(依赖严格单向:上层引用下层,下层不知上层)

```
L4 Research(判断:观象/坩埚)     L4 Execution(执行:观澜)
        └── Judgment / Decision(桥)──┘
L3 Audit    Experiment / Outcome / Lineage / 兑现审计
L2 Compute  跨平台公共统计,唯一实现
L1 Data     qbase:Entity Master + Time&PIT / Fact / Signal
```

## 六对象(全体系统一数据契约)

| 对象 | 一句话 |
|---|---|
| Fact | 市面可重拉的公共事实(qbase 存) |
| Signal | 自建口径加工产物(各平台产,qbase 视图归一读) |
| Judgment | 系统或人的判断,必须带有效期,不许永久悬置 |
| Decision | 人实际做了什么(含未执行的念头) |
| Experiment | 每次假设检验档案(淘沙台账) |
| Outcome | 兑现结果(记分/对账/过期未验证) |

## 六平台归位与当前姿态

| 平台 | 归位 | 姿态(本机视角) |
|---|---|---|
| qbase | L1 底座 | **本机在建**:回填+归一视图。统一读法不统一存储——自产数据原库不动,视图去读 |
| 淘沙 | L2+Experiment | **本机在建**:事件研究引擎+假设台账 |
| 雷达 | 发现+对账(将拆两刀入L1/L3) | 老机无人值守,**只经只读视图读它,零改动** |

| 坩埚 | L4 Research | 老机只运行,等判决日,同上只读 |
| 观澜 | L4 Execution | 老机待命,不接触 |

## 本机三条最高纪律

1. **对老平台一切只读**,qbase 凭据不出本机;2. **双时戳+append-only+含退市票**,回填的 observed_time 如实标批次;3. **holdout 线(2024-07-01)焊在视图定义里**,任何代码不得绕开。
